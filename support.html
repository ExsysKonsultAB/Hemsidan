Support

För bästa support ber vi er vänligen skapa ett ärende via vårt ärendehanteringssystem. Vi har bevakning av ärenden under dagtid och återkommer så fort vi kan med uppdaterad information.

Öppna nytt ärende

Ange så mycket information som möjligt så att vi kan bäst hjälpa dig. För att uppdatera ett tidigare lämnat ärende, vänligen logga in.

Öppna ett nytt ärende

Kontrollera Ärende Status

Vi tillhandahåller arkiv och historia med alla dina nuvarande och tidigare supportfrågor komplett med svaren.

Fjärrsupport

Vid support ärenden eller andra tillfällen kan vi behöva fjärrstyra datorer hos er. Vi använder vi ett program som heter TeamViever. Klicka på länken nedan förr att starta fjärrsupport, ange det ID som visas när programmet startas!
